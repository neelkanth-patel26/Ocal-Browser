OCAL BROWSER - RELEASE NOTES v1.1.0
=================================

Release Date: March 27, 2026
Version: 1.1.0
Codename: "Crystal Stream"

Ocal Browser v1.1.0 brings a faster, more secure, and polished browsing experience. This update focuses on a streamlined interface, a brand-new automated update engine, and critical security and stability fixes.

1. UI & USER EXPERIENCE REFINEMENTS
-----------------------------------
- [NEW] Sidebar Tab Simplification: The "Settings" tab has been removed from the sidebar navigation strip for a cleaner, more focused experience.
- [REFINED] Persistent Settings Access: The settings gear icon is now a permanent fixture in the sidebar header, providing instant access to full preferences without cluttering the tab strip.
- [NEW] Glassmorphic Window Controls: Fully functional and aesthetically matched window controls (Minimize, Maximize, Close) with Windows 11-native effects.
- [NEW] Pill-style Tab Aesthetic: Active tabs now feature a modern pill-shaped design, improving visual hierarchy.

2. MODERNIZED UPDATE SYSTEM
---------------------------
- [NEW] Background Streaming Updates: Ocal now downloads updates silently in the background, minimizing interruption to your workflow.
- [NEW] Real-time Progress Tracking: View exact numeric progress during the download phase for better transparency.
- [NEW] Architecture-Aware Selection: The update engine automatically selects the correct binary for your system (x64), ensuring optimal performance.
- [NEW] Resilient Retry Logic: Improved download stability with automatic retries on network failure.
- [NEW] Silent Installation: Updates are now installed seamlessly via an improved Inno Setup engine.

3. SECURITY & COMPATIBILITY
---------------------------
- [SECURITY] Stealth Network Configuration: Refined network headers and browser fingerprinting to prevent automated detection and improve privacy.
- [FIX] Google Services Authentication: Resolved "Not Secure" blocks on Google/YouTube login flows.
- [FIX] OAuth Popup Restoration: Restored native window behavior for third-party authentication popups.
- [STABILITY] IPC Communication Optimization: Fixed critical messaging errors between the renderer and main process for smoother navigation.

4. BUG FIXES & STABILITY
------------------------
- FIXED: Address bar synchronization issues when switching between multiple tabs.
- FIXED: Redundant "Settings" entries in the sidebar that caused duplicate rendering.
- FIXED: "Unknown Publisher" warnings for the Windows installer through improved signing alignment.
- IMPROVED: Cache management for native dependencies during the build process.
